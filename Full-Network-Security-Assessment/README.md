# Full Network Security Assessment

## Objective

Conduct an end-to-end network security assessment using industry-standard tools.

## Tools

- Nmap
- Wireshark
- Nikto
- DVWA
- BurpSuite
- Sqlmap
- Windows 11
- WSL Ubuntu

## Assessment Phases

### Phase 1

Network Reconnaissance

- Host discovery
- OS detection
- Service detection

### Phase 2

Traffic Analysis

- HTTP
- DNS
- TCP
- ARP

### Phase 3

Web Vulnerability Assessment

Nikto identified several configuration weaknesses including:

- Missing security headers
- Directory indexing
- HTTP TRACE
- Information disclosure
