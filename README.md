# 🛡️ OIBSIP Cyber Security Internship

This repository contains the projects completed as part of the **Oasis Infobyte (OIBSIP) Cyber Security Internship**. The tasks cover fundamental cybersecurity concepts including network scanning, firewall configuration, vulnerability assessment, web application security, packet analysis, and security research.

---

## 📌 Internship Overview

The internship focuses on developing practical cybersecurity skills through hands-on projects and research assignments.

### Skills Demonstrated

- Network Scanning
- Firewall Configuration
- Web Application Security
- SQL Injection Testing
- Vulnerability Assessment
- Packet Analysis
- Network Security Research
- Patch Management
- Security Reporting

# ⚠️ Ethical Statement

All practical exercises were performed **only in a controlled local laboratory environment** using intentionally vulnerable applications (such as DVWA) or systems owned by the author.

No testing or scanning was performed against public websites, production environments, or systems without explicit authorization.
