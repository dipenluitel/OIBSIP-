# Common Network Security Threats

## Project Overview

This repository contains a research report on common network security threats. The report explains how major network attacks work, their real-world impact, examples of known incidents, and effective mitigation strategies used by organizations to protect their networks.

## Objective

The objective of this project is to:

* Understand common network security threats.
* Learn how these attacks affect organizations.
* Explore practical mitigation strategies.
* Summarize the threats in a comparison table.

## Report Contents

* Introduction
* DoS/DDoS Attacks
* Man-in-the-Middle (MITM) Attacks
* IP Spoofing
* DNS Poisoning/Spoofing
* Comparison Table
* Conclusion
* References
