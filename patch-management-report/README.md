# Task 6 – Research Report: Patch Management and Software Updates

## Project Overview

This repository contains a research report on patch management and software updates. The report explains why patch management is important, how vulnerabilities are identified and tracked, the consequences of poor patch management, and best practices for maintaining secure systems.

## Objective

The objective of this project is to:

* Understand the importance of patch management.
* Learn how vulnerabilities are identified using CVEs and CVSS scores.
* Analyze a real-world cybersecurity incident caused by missing security patches.
* Explore industry best practices for maintaining secure systems.

## Report Contents

* Introduction
* What is Patch Management?
* Patch Management Lifecycle
* WannaCry Ransomware Case Study
* CVE and CVSS Overview
* Patch Management Best Practices
* Conclusion
* References
